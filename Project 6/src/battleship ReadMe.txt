battleship.java Read Me

This program will produce a non working model for the game battleship.

The program will consist of three layouts:
	-First layout is the GridPane which will take care of the general board, which
	consists of Text output of hits, misses, sunk and the buttons for the board
	itself.
	-Second layout is the HBox which will allow the context text to be placed on top.
	This textbox will output the context of the game currently
	-Third is the BorderPane which will control the regions which will exist. The 
	HBox context label will go top region, while the GridPane nodes will go center.
